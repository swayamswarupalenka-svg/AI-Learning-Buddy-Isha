# 🎓 AI Learning Buddy

An AI-powered learning assistant built using **Streamlit** and **Google Gemini API**.

## 📌 Features

- 📖 Explain any topic in simple language
- 🌍 Provide real-life examples
- 📝 Generate a 5-question quiz with answers
- 💬 Ask anything about a topic

## 🛠 Technologies Used

- Python
- Streamlit
- Google Gemini API
- Google Colab
- ngrok

## 🚀 How to Run

1. Clone this repository.
2. Install the dependencies:

```bash
pip install -r requirements.txt
```

3. Add your own Gemini API key in `app.py`:

```python
genai.configure(api_key="YOUR_GEMINI_API_KEY")
```

4. Run the application:

```bash
streamlit run app.py
```
